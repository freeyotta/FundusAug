# 眼底图像增强工具说明 (FundusHypothesisAug)

这是我为眼底图像研究写的一个数据增强工具包。

### 1. 怎么安装依赖？
你可以直接通过项目中的环境核对表进行一键安装。建议在虚拟环境中运行：

```bash
# 使用 pip 一键安装所有依赖库
pip install -r requirement.txt
```

如果手动安装，核心依赖包括：
`pip install opencv-python albumentations scipy matplotlib Pillow`

### 2. 这个包里有什么？

* **基础增强**：包括旋转、平移、亮度对比度调整、模糊、低分辨率模拟等常用操作。
* **物理模拟 (核心)**：这部分参考了 **TMI 论文的官方源码**，专门用来模拟**白内障**导致的视觉退化。它利用距离场和通道反转散射逻辑，还原了真实的物理退化过程。

### 3. 如何快速看效果？

1. 在项目文件夹里放一张叫 `test_img.jpg` 的原始眼底图。
2. 在终端运行测试脚本：`python AugTestBench.py`。
3. 运行完后，文件夹里会自动生成一张包含 6 种模式对比的图 `result.jpg`。

### 4. 怎么在代码里用？

你可以参考下面的例子快速集成到你的训练流程中：

```python
from FundusHypothesisAug import FundusHypothesisAug
import cv2

# 1. 初始化增强引擎
# 指定 img_size 确保输出尺寸一致，默认为 (512, 512)
aug = FundusHypothesisAug(img_size=(224, 224))

# 2. 读取图像 (OpenCV 读取默认为 BGR)
image = cv2.imread("test_img.jpg")

# 3. 调用增强接口
# mode 可以根据研究需求选择不同的假设空间变换
result = aug.forward_augmentation(image, mode="Cataract")

```
## 🌟 核心特性 (Features)

* **空间结构形变 (`Geometry`)**：包含随机旋转、平移、缩放以及三维透视投影形变，完美模拟拍摄时光轴未对准瞳孔中心的非正轴歪斜。
* **硬件传感器波动 (`Color`)**：涵盖亮度对比度微调、饱和度漂移及 Gamma 校正，模拟跨医院、跨厂家相机传感器的显色差异。
* **临床镜头伪影 (Lens Shadows & Artifacts)**：
  * **全局高斯模糊 (`Blur`)**：采用纯净无噪点的大内核高斯模糊，强力模拟相机全图对焦失败（失焦）或镜头起雾。
  * **运动抖动模糊 (`MotionBlur`)**：带有方向延展性的运动模糊，模拟患者瞬目（眨眼）、不自主眼球震颤或拍摄时的物理晃动。
  * **低分辨率重采样 (`LowRes`)**：下采样最近邻插值（故意制造马赛克锯齿）再通过双三次插值拉伸，模拟网络远程传输压缩与低端设备的低画质成像。
* **物理/光学仿真引擎 (Advanced Simulation)**：
  * **过曝模拟 (`Overexposure`)**：**基于二维高斯分布矩阵**生成平滑衰减的光斑 Mask，通过 `np.clip` 截断像素值。完美仿真由于光源过强或瞳孔未完全散大引起的角膜圆形反光斑，模拟特征不可逆丢失。
  * **局部阴影 (`Shadow`)**：利用高斯反转 Mask 对图像局部进行点乘变暗，边缘具备自然的“半影区”渐变过渡，完美仿真眼睑（眼皮）遮挡、睫毛伪影或外周局部光照不均。
  * **白内障物理模拟 (`Cataract`)**：100% 还原 **TMI (IEEE Transactions on Medical Imaging) 顶级期刊论文官方物理算子**，利用距离场变换（Distance Transform）与通道反转散射逻辑，真实模拟晶状体浑浊对光线的侵蚀。
* **解耦随机流水线 (`Mixed`)**：全新重构的复合退化流水线。各功能层采用松散的低概率门控独立触发，参数在连续区间内随机波动，能生成极其丰富的现实复合噪声组合。

---
####  参数说明表

| 参数名 (Parameter) | 类型 | 默认值 | 说明 (Description) |
| --- | --- | --- | --- |
| **`img_size`** | `tuple` | `(512, 512)` | 初始化参数。定义输出图像的分辨率 (H, W)。 |
| **`image`** | `ndarray` | - | 输入图像。支持 OpenCV 格式 (H, W, C)，建议为 BGR 顺序。 |
| **`mode`** | `string` | `"Mixed"` | **增强模式选择**。决定图像进入哪种物理或几何假设空间。 |

####  可选模式 (`mode`) 详解

| 模式字段 (`mode`) | 底层算法与核心算子 | 对应真实临床/物理退化场景 |
| :--- | :--- | :--- |
| **`"Original"`** | 图像不做任何增强处理，直接原样返回。 | **实验基准对照样本 (Baseline)**：用于多中心鲁棒性验证或消融实验的纯净样本控制。 |
| **`"Geometry"`** | 仿射变换 + 三维透视投影形变。<br>主要算子：`A.ShiftScaleRotate`, `A.Perspective` | **拍摄角度歪斜与空间位移**：模拟操作医生在对准瞳孔时产生的物理偏差，以及患者歪头、眼球转动导致的非正轴斜切拍摄。 |
| **`"Color"`** | 随机亮度对比度波动 + 饱和度漂移 + Gamma校正。<br>主要算子：`A.RandomBrightnessContrast`, `A.HueSaturationValue`, `A.RandomGamma` | **多中心硬件传感器色差**：模拟不同医院、不同厂家眼底相机（如不同型号 CCD 传感器）由于出厂增益、光源老化导致的图像显色与色调差异。 |
| **`"Blur"`** | 强力、纯净的大内核高斯模糊。<br>主要算子：`A.GaussianBlur(blur_limit=(35, 55))` | **设备失焦与对焦失败**：模拟相机光学对焦系统失效、散焦，或者镜头表面大面积起雾、落尘带来的全局细节退化。 |
| **`"LowRes"`** | 空间下采样（最近邻插值）+ 强行双三次插值拉伸。<br>主要算子：`A.Downscale`, `A.Resize` | **低端成像与网络压缩传输**：故意制造图像锯齿与马赛克感。模拟基层医疗机构低像素陈旧设备成像，或互联网医疗跨中心传输时的流媒体有损压缩丢包。 |
| **`"MotionBlur"`** | 具备方向延展性的运动模糊。<br>主要算子：`A.MotionBlur(blur_limit=(25, 45))` | **患者瞬目与瞬时晃动**：真实模拟患者在闪光灯亮起瞬间不自主地眨眼（瞬目）、眼球震颤，或者由于拍摄瞬间物理晃动带来的伪影。 |
| **`"Overexposure"`** | **二维高斯光斑叠加模拟 + 饱和度截断**。<br>底层逻辑：`_simulate_overexposure_core` | **角膜强反光斑（信息丢失）**：基于物理机制，随机生成中心向四周衰减的高斯高亮光斑，并使用 `np.clip` 强行阻断。模拟光源过强或瞳孔放得不够大时，永久抹去局部视网膜血管特征的强反光区域。 |
| **`"Shadow"`** | **二维高斯反转矩阵 + 局部像素点乘变暗**。<br>底层逻辑：`_simulate_shadow_core` | **外周遮挡与照明不均（半影区）**：生成边缘过渡极其平滑的局部变暗层。完美模拟眼睑（眼皮）边缘遮挡、睫毛伪影或光源角度偏斜产生的弱光暗角。 |
| **`"Cataract"`** | **TMI 官方物理仿真算子**。<br>底层逻辑：距离场变换（Distance Transform） + 通道反转散射。 | **晶状体病理级浑浊**：100% 还原 IEEE TMI 顶级期刊论文官方物理逻辑，真实模拟光线穿过浑浊晶状体时的光学散射，造成眼底特有的“白茫茫”雾蒙蒙侵蚀。 |
| **`"Mixed"`** | **多层解耦随机流水线（概率门控级联）**。<br>底层逻辑：各噪声层以独立低概率触发。 | **极复杂的现实复合噪声假设空间**：模拟临床中最棘手的复合退化——如一个患有白内障的患者在配合度低（发生旋转）且带有瞬时眼球抖动和遮挡时的综合样本空间。 |

> **注意 (Tips)**:
> * `forward_augmentation` 返回的是 `numpy.ndarray` 格式。
> * 如果用于 **PyTorch** 训练，请确保在增强后使用 `transforms.ToTensor()` 将其转换为张量，并最后进行标准归一化。
> * **白内障模式**（Cataract）会对图像色彩产生较大物理扰动，建议仅在训练阶段开启。
